# Literature Studies

This directory contains literature studies and references for the 2026 research proposal.

## Structure

- `papers/` - PDF files of research papers and articles
- `notes/` - Literature review notes, summaries, and annotations
- `bibliography.bib` - BibTeX bibliography file
- `literature-matrix.md` - Comparison matrix of key papers

## Guidelines

### File Naming Convention
- Papers: `AuthorYear_ShortTitle.pdf` (e.g., `Smith2024_MachineLearning.pdf`)
- Notes: `AuthorYear_notes.md` (e.g., `Smith2024_notes.md`)

### Organization Tips
- Group related papers by theme or methodology
- Use consistent citation keys in bibliography
- Link notes to corresponding papers
- Update the literature matrix regularly

## Bibliography Management

The `bibliography.bib` file contains all references in BibTeX format. Use citation keys that match the file naming convention for easy cross-referencing.
